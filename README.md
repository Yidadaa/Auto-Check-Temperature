**提示：Github Releases 服务在国内属于被墙状态，所以建议使用<a href="https://gitee.com/yidadaa/Auto-Check-Temperature/releases">国内镜像仓库</a>下载安装包。**

## 功能
1. 自动填充账号密码（完全本地存储，具体可看源码）；
2. 一键报平安和三次体温打卡，节省宝贵生命，有这时间为啥不去摸鱼呢（呲牙）。
